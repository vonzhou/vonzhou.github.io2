使用hexo构建博客

Posted on 2015-10-20   |   In Tech

1.安装git和node.js，npm

2.安装hexo： npm install -g hexo-cli， 在ubuntu12.04上会出现问题，参考这里

3.安装hexo成功后，可以在本地构建博客系统

4.可以启动本地服务，看到博客的状态了： hexo server

5.可以安装自己喜欢的主题，http://hexo.io/themes/ 有很多，比如选择next

6.在_config.yml中配置语言和主题

7.可以hexo new page对应的菜单，如标签，归类，个人主页等等，然后修改theme/next/_config.yml中的menu项

8.配置部署选项，这里使用github
